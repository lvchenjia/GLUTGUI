# GLUT GUI
### Introduce
Use glut to implement a window with button and textbox

使用glut库实现了窗口以及按钮、文本框两个组件

组件有点击、焦点改变、鼠标悬停、文本输入等事件

实现了GUI的基本功能

### How to compile
```bash
g++ main.cpp -lGL -lglut -lGLU -o main
```
